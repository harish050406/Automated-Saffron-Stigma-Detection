# Automated Saffron Stigma Detection

Using YOLOv8-segmentation to automate saffron harvesting by pinpointing the exact plucking spots for robotic tools.

## 🛠️ Project scripts included:
* `train_model.py` - Core training pipeline
* `validate_model.py` - Performance validation script
* `geometry_engine.py` - Central centroid calculation matrix
