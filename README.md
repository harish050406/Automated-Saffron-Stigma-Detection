# Automated-Saffron-Stigma-Detection
Problem:Harvesting saffron by hand is exhausting, costly, and easily ruins delicate flowers. Solution: We used YOLOv8-segmentation to automate this, letting AI instantly pinpoint the exact plucking spots for robotic tools. How it works: Saffron photos are processed on a GPU where the AI maps out thin stigmas to find precise cutting coordinates.
